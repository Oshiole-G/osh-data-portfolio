# Power BI Dashboards & Reporting

This folder contains Power BI dashboard projects focused on business reporting, KPI tracking, and data visualization.

## Contents
- Data import and transformation (Power Query)
- Data modeling and relationships
- DAX measures and calculated columns
- Interactive dashboards (slicers, filters, drill-down)
- KPI reporting and performance analysis
- Sales and lead funnel dashboards

## Tools Used
- Power BI Desktop
- Power Query
- DAX

## Notes
Each project includes the PBIX file (where possible), screenshots of dashboards, and a short summary of insights.
